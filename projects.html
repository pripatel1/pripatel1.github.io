Amazon Tracking Center: Optimization Model for Investment, Shipping, and Routing

An advanced system designed to optimize Amazon's tracking operations, including investment strategies, shipping logistics, and routing efficiency.

Login System

The project features a secure login system with role-based access control:

  • Users can log in as either a truck driver or a manager.
  • The system validates usernames, passwords, and user types.
  • Error messages are displayed for invalid credentials or mismatched user types.

User Navigation

After successful login, users are presented with a navigation screen:

  • Options include 'View Investment,' 'View Shipping,' and 'View Routing'.
  • Access to certain features is restricted based on user role (e.g., truck drivers cannot access investment data).

Investment Optimization

Managers have access to an investment optimization tool:

  • Allows for minimizing expected risk and maximizing profit.
  • Provides detailed information on expected return, risk, and profit.

Shipping Optimization

Both managers and truck drivers can access the shipping optimization feature:

  • Users can prioritize specific cities for shipping.
  • The system calculates an optimal solution and updates total cities and percentages.

Routing Optimization

The routing page offers advanced path-finding capabilities:

  • Users can input start and end cities, along with optional midpoint stops.
  • The system calculates the most efficient total path.

Admin Features

A hidden admin page stores login credentials for both managers and truck drivers, ensuring secure access management.
